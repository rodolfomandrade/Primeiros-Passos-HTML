Como surgiu a Internet

A internet surgiu por uma necessidade militar dos Estados Unidos de garantirem que, em caso de uma instalação militar fosse atacada, os documentos que ali estavam, como por exemplo planos sobre o inimigo ou projetos de tecnologia não seriam perdidos, surgindo assim a Arpanet ( Advanced Research Projects Agency Network ) que teve a primeira conexão realizada em 29 de outubro de 1969, entre a Universidade da Califórnia e o Instituto de Pesquisa de Stanford.

Na década de 90, surgiu um cientista chamado Tim Berners-Lee que criou a World Wide Web ( WWW ) e assim deu a todos a oportunidade de utilizar a internet por meio de um navegador que ele mesmo criou, fazendo assim a a cultura atual de acesso a internet que temos hoje em dia.

Diferença entre Cliente e Servidor

Em uma rede, seja ela Mundial como a internet, ou uma rede local como em uma lan house ou uma empresa, o Servidor tem o papel de oferecer algum serviço, seja um site, seja como armazenamento de arquivos, seja como um servidor de vídeos tal como youtube, amazon, netflix e etc..., como servidor de e-mail ou outros serviços, ele também pode ser utilizado para gerenciar os acessos de rede, ou até para autenticar quem está utilizando ou quem pode utilizar aquela determinada rede, serviço ou mesmo que site pode ser acessado. Servidores costumam ser máquinas pesadas e caras, pois por ter mais poder de processamento, precisam de mais refrigeração e fontes de alimentação mais parrudas.

O Cliente é o computador, Notebook, celular, tablet ou outro dispositivo que se conecta com o Servidor para utilizar os serviços deste, então o Cliente sempre será operado por um usuário que quer utilizar algum tipo de serviço, seja um streaming, um site ou mandar e receber um e-mail. Costumam ser equipamentos mais modestos e com processamento mais fraco mas que é potente o suficiente para conseguir acessar aquele serviço que está em um servidor.

O que é uma linguagem de Programação

É uma linguagem formal, que através de uma série de instruções, permite que você dê ordens ao computador para que ele siga uma sequência de ações e isso gere um resultado, seja resolvendo um problema, seja permitindo controlar uma máquina, seja gerando um resultado visual e assim a pessoa possa interagir com esse resultado, basicamente um programador é o chefe e o computador é o funcionário que pode fazer muitas coisas, desde que o chefe saiba dar a sequência correta de ordens.

Existem alguns tipos de linguagem de programação

  • Linguagem de máquina: é a linguagem base, a que a máquina entende pois é formada apenas de binários e como uma máquina só entende 0 e 1 que seria ligado e desligado, então é a linguagem mais próxima do nível da máquina.
  • Linguagem Assembly: é uma linguagem criada para que fosse mais fácil dar instruções a máquina, já que não são somente instruções em binário.
  • Linguagens de alto nível: são linguagens mais próximas da lingua humana, facilitando assim de se fazer programas pois, se consegue dar ordens como se estivesse falando com outro ser humano.

Existem diversas ferramentas que são utilizadas quando se escreve um código, sejam eles compiladores ( que converter a linguagem humana para linguagem de máquina ), Editores de códigos que ajudam a organiza-lo e a fazer comentários entre outras funcionalidades, temos também os interpretadores que são basicamente um tradutor que traduz o código em tempo real para a máquina, o código que um humano escreveu.

Entendendo o HTML e sua Estrutura Base

O HTML não é uma Linguagem de programação como muitos devem achar, pois não se trata de uma linguagem com uma lógica a ser realizada, ela é simplesmente uma linguagem de marcação de texto, dando sentido assim as páginas da web mas sem alterar sua interatividade ou interação.

Apesar disso, ela é essencial para se fazer um site, pois é ela que dará sentido e estruturará a página da web, é composta de tags semânticas, algumas que tem abertura e fechamento, outras não precisam ser fechadas, ele irá conter todo o conteúdo de um site.

O HTML foi inventado por Tim Berners-Lee, para criar um sistema de hipertexto na internet, facilitando assim a navegação pela mesma e podendo ser disponibilizado mais material para consumo, HTML significa HYPERTEXT MARKUP LANGUAGE ou em português Linguagem de Marcação de Hipertexto
